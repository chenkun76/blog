## 框架：Nuxt 项目

```bash
# install dependencies
# 下载项目依赖的包
$ npm install

# serve with hot reload at localhost:3000
# 启动项目，端口号3000，模式为热加载
$ npm run dev

# build for production
# 构建项目，生产文件
$ npm run build

# build for launch server
# 构建项目，运行服务器
$ npm run start

# generate static project
# 生产静态项目
$ npm run generate
```
